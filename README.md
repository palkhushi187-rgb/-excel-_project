# Excel Sales Dashboard Project

## 📊 Project Overview
This project is an interactive **Excel Dashboard** created to analyze sales data.  
The dashboard helps users quickly understand sales performance, trends, and key metrics using visualizations.

## 🚀 Features
- Interactive Excel Dashboard
- Sales performance analysis
- Dynamic charts and graphs
- Data filtering for better insights
- Clean and easy-to-understand layout

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## 📂 Dataset
The dataset contains sales information such as:
- Product Name
- Sales Amount
- Region
- Date
- Customer Details

## 📈 Dashboard Insights
The dashboard shows:
- Total Sales
- Top Performing Products
- Sales by Region
- Monthly Sales Trends
